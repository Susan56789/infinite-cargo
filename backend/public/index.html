Infinite Cargo API

Secure, Scalable, Reliable APIs for Freight Management

API Base URL

https://api.infinitecargo.co.ke/api (or http://localhost:5000/api in development)

Key Endpoints

GET /api/health — Check API health
POST /api/auth/login — Authenticate user
GET /api/users — Get list of users
GET /api/trucks — Get all trucks
GET /api/loads — View posted loads
GET /api/bids — See all bids
GET /api/bookings — View bookings
